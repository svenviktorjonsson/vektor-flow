# Chess Piece Meshes

Source: OpenGameArt `3d chess pieces` by tunakron

URL: https://opengameart.org/content/3d-chess-pieces

License: CC0

The GLB files in `chess_pieces/gltf` are preserved as source assets. The VKF
runtime example uses baked mesh literals in `../../lib/piece_meshes.vkf` so the
program can run without a web asset loader or chess-specific JavaScript.
